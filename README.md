# CSD: LLM-Based Detection of Scam and Phishing Messages Using Public Datasets

This repository contains the notebook-based analysis pipeline for phishing and scam email detection using the CEAS_08 and Enron public datasets.

## Project Structure

- `00_setup_and_config.ipynb`: project paths, dependencies, and setup notes.
- `01_merge_and_describe.ipynb`: dataset loading, merging, and descriptive statistics.
- `02_preprocess_and_split.ipynb`: text cleaning, deduplication, and stratified train/validation/test splitting.
- `03_train_ml_baselines.ipynb`: TF-IDF and Word2Vec traditional machine learning baselines.
- `04_openai_embeddings_experiments.ipynb`: optional OpenAI embedding experiments.
- `05_champion_robustness_and_errors.ipynb`: champion model comparison, robustness testing, and error analysis.
- `06_academic_figures_and_tables.ipynb`: dissertation-ready figures and summary tables.
- `00_run_all_pipeline.ipynb`: notebook-only full pipeline runner.

## Output Policy

All generated results are written under:

```text
output/
```

Small dissertation-ready outputs such as figures, reports, and statistics can be stored in the repository. Large generated artifacts are intentionally excluded from GitHub and can be regenerated by running the notebooks locally:

- `output/data/`
- `output/embeddings/`
- `output/models/`
- `output/error_analysis/`
- `output/reference_preview/`

## API Key

OpenAI experiments read the API key from:

```text
openai_api_key.txt
```

This file is intentionally excluded from GitHub. Create it locally and paste only the key value inside the file when running notebooks 04 and 05.

## Recommended Run Order

1. `00_setup_and_config.ipynb`
2. `01_merge_and_describe.ipynb`
3. `02_preprocess_and_split.ipynb`
4. `03_train_ml_baselines.ipynb`
5. `04_openai_embeddings_experiments.ipynb` if OpenAI API access is available
6. `05_champion_robustness_and_errors.ipynb`
7. `06_academic_figures_and_tables.ipynb`

.venv/bin/python
